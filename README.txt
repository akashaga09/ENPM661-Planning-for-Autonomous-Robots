Instructions to run the program:-

1. Run the python script.
2. Enter the numbers column wise, separated by a space.
	For example - 1 4 7 2 5 8 3 6 0 (column wise entry for the goal matrix ----> 1 2 3
										     4 5 6
										     7 8 0)
										     
PS - The first row in each output file generated represents the column index and should be ignored.
